What is this page for?

This page allows you to create or provide a configuration file for your machine learning experiment. The configuration file contains all the settings needed to run your experiment.

Available Options:
Option 1: Upload

Upload an existing configuration file (.yml, .json) that you already have.

Option 2: Generate Template

Create a basic template file that you can download and edit manually.

Option 3: Template Optimizer

Use our advanced wizard to create a comprehensive configuration through guided steps.

Tip: If you're new to machine learning experiments, we recommend using Option 3 (Template Optimizer) for the best guided experience.
