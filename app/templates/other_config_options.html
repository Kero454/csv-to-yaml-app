What are these options?

These are alternative methods for creating configuration files that offer different levels of guidance and complexity.

Available Options:
Option 3: Basic Form Editor

Best for: Users who want a simple, guided form interface

Features:

  • Step-by-step form interface
  • Field validation
  • Basic configuration options
  • Beginner-friendly
Option 4: Use Template

Best for: Users who want pre-configured settings

Features:

  • Pre-filled default values
  • Time series experiment focus
  • Complete configuration coverage
  • Ready-to-use settings
Recommendation: If you're looking for maximum control and customization, try Option 3 (Template Optimizer) from the main page. For quick setup with good defaults, use Option 4 here.
